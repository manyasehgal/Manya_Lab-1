# Manya_Lab-1

# Lab 1: Machine Learning Metrics and Regression

## Overview
This repository demonstrates key machine learning and regression concepts using Python in a calculator-style approach. Each problem is solved step-by-step with clear code, outputs, and explanations, making it easy to follow and reproduce.

---

## Contents

### Problem 1: Confusion Matrix Metrics
- Binary classification predicting flood occurrence.
- Metrics: Accuracy, Precision, Recall, F1 Score.
- Includes a bar chart visualization of metrics.

### Problem 2: Coefficient of Determination (R²)
- Regression model predicting river discharge.
- Calculation of SSR, SSE, SST.
- Interpretation of R² regarding variance explained.

### Problem 3: Mean Squared Error (MSE)
- Computation of MSE from squared errors of predictions.
- Demonstrates error evaluation in regression models.

### Problem 4: Cook's Distance
- Identification of influential observations in regression.
- Calculation of Cook’s Distance and comparison with threshold.

### Problem 5: Simple Linear Regression Prediction
- Regression equation: y = 2.5x + 10.
- Prediction of crop yield based on rainfall.
- Explanation of slope and intercept meaning.

---

## Results Summary
- Confusion Matrix Metrics: Accuracy = 0.82, Precision ≈ 0.86, Recall ≈ 0.83, F1 Score ≈ 0.85
